# Rock Paper Scissors Game

This is a simple Rock Paper Scissors game built using HTML, CSS and JavaScript.

The user selects rock, paper or scissors by clicking a button.
The computer randomly chooses one option. The result is displayed instantly and the score updates after each round.

## Features

* Interactive buttons for user choice
* Random computer selection
* Real time result display
* Live score tracking
* Simple and clean interface

## What I Learned

While building this project, I practiced:

* DOM manipulation
* Event listeners
* Writing conditional logic
* Using Math.random()
* Updating content dynamically

This project helped me improve my understanding of JavaScript logic and how to update the UI based on user interaction.

## Technologies Used

* HTML
* CSS
* JavaScript

## How It Works

* The user clicks one of the buttons
* The computer selects a random choice
* The game compares both selections
* The result is displayed
* The score updates depending on who wins

## How to Use

1. Open the project in your browser
2. Click rock, paper or scissors
3. See the result instantly
4. Keep playing and track the score
